## What's MicroRegEx
MicroRegEx is a micro regular expression engine.

## Operator
* `*` - zero or more repetitions
* `+` - one or more repetitions
* `?` - optional
* `a|b` - matches a or b
* `(expr)` - treat the `expr` as an atom
* `\` - escape characters
